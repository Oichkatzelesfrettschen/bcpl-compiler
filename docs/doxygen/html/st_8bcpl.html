Detailed Description

BCPL compiler main driver and syntax analysis.

Author
Robert Nordier
Date
2012

This is the main compiler module that handles command-line options, initialization, and coordinates the compilation phases including lexical analysis, syntax analysis, and code generation.

Key sections:

  • MASTER: Main entry point and option processing
  • LEX: Lexical analyzer for tokenizing BCPL source
  • SYN: Syntax analyzer for parsing BCPL grammar
  • TRN: Code generation and optimization

Compilation options supported:

  • P: Parse only (syntax check)
  • T: Tree dump for debugging
  • C: Code generation
  • M: Memory map
  • N: No optimization
  • S: Print source code
  • E: Enable lexical tracing
  • L: Set tree size limit
  • 3: Use 3-word save space
