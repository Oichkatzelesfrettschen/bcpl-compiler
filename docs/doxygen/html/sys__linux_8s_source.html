7 * Provides Linux-specific system call wrappers for the BCPL runtime system.
8 * Maps BCPL I/O and system operations to Linux system calls using the
9 * standard Linux x86 system call interface.
10 *
11 * System calls supported:
12 * - File I/O: read, write, open, close, lseek
13 * - Process control: exit
14 * - Memory management: brk
15 * - Error handling: Standard errno conventions
16 *
17 * @see sys.s for the generic system call dispatcher
18 * @see su.s for the startup code that uses these interfaces
19 */
